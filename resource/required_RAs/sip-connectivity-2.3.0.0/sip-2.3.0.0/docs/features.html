RA Features

This section describes additional features that are particular to the OpenCloud SIP RA implementation.

Cluster Support

This section describes how the SIP RA can be used in a cluster

Replicated Dialogs

The SIP RA can support replicated dialog activities, so that dialogs can be continued on other nodes in a cluster after a node failure.

Forking

This section describes how applications can use the SIP RA to handle forking scenarios.

Lazy Parsing

This section describes the RA's lazy parsing implementation, and how applications can handle non-compliant SIP user agents.

Outbound Connections

This feature allows the SIP RA to maintain persistent outbound connections to a host on the other side of a firewall or NAT.
